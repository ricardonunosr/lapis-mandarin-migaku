# Lapis Mandarin Migaku

An advanced, lightweight Anki notetype for Mandarin Chinese, adapted from the original Japanese Lapis notetype to work seamlessly with Migaku.

### Why use Lapis Mandarin Migaku?
- **High Performance:** Minimal JavaScript ensures fast, lag-free reviews on both desktop and mobile devices.
- **Clean Layout:** Automatically organizes your definitions, sentences, and media into a highly scannable interface.
- **Migaku Integration:** Optimized to support Migaku's parsing, audio, and definition formatting without breaking.

### Quick Setup
1. Download the example deck from the [Releases](https://github.com/ricardonunosr/lapis-mandarin-migaku/releases) page.
2. Map your Migaku export fields to match the Lapis fields (Expression, Meaning, Audio, Sentence).
3. Customize your font sizes and dark/light mode colors directly at the top of the card's **Styling** CSS section.
